Área do Sistema
 
1. Área de Usuários (Identidade)
Gerenciar tudo o que envolve o acesso ao sistema. Inclui processos de cadastro, login, atualização de perfis e a definição do papel do usuário (aluno ou professor).
2. Área de Conteúdo (Catálogo de Mídias)
Funciona como o "caderno" virtual. Gerencia as fotos das lousas, os vídeos das aulas e os arquivos em PDF. O foco principal é armazenar e exibir o material educacional.
3. Área de Interação
Administrar a comunicação e a avaliação dos conteúdos. Engloba o registro de comentários e a atribuição de notas para as aulas
4. Área de Gameficação
Inclui a área de ranking e a distribuição de pontos para alimentar o sistema de classificação (gamificação).
